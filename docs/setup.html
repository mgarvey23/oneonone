One-time setup

This page helps you scramble your GitHub token before pasting it into app.js, so it doesn't get auto-revoked by GitHub's secret scanner.

  1. Create a fine-grained personal access token at github.com/settings/personal-access-tokens/new:
    • Repository access: Only select repositories → oneonone
    • Repository permissions → Contents: Read and write
    • Generate, copy the token (starts with github_pat_)
  2. Paste the token below and click Scramble:
